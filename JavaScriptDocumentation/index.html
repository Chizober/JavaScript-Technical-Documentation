Introduction

JavaScript is a lightweight,dynamic programming language used to make web pages interactive.It is also fully integrated with HTML and CSS. It is supported by all major browsers and enabled by default. Although Java and JavaScript are similar in name, syntax, and respective standard libraries, the two languages are distinct and differ greatly in design.

Parts of Javascript

JavaScript is made up of the following three distinct parts:

  • The Core (ECMAScript)

    • ECMAScript,its extensions and implementations exist on Web browsers,NodeJS(a server-side JavaScript platform) and Adobe Flash. Extensions, such as the Document Object Model (DOM), use ECMAScript’s core types and syntax to provide additional functionality to the environment.It is a base upon which more-robust scripting languages may be built. ECMA-262 describes the following parts of ECMAScript:

    • Syntax
    • Types
    • Statements
    • Keywords
    • Reserved words
    • Operators
    • Objects

    The different versions of ECMAScript are defined as editions,and these different versions,introduced additional functionalities which include:

  • Native JSON object for parsing and serializing JSON data
  • Methods for inheritance and advanced property definition
  • New statements and data structures
  • True classes and classical inheritance
  • A new strict mode that slightly augments how ECMAScript engines interpret and execute code
  • String handling
  • Support for regular expressions etc
  • The Document Object Model (DOM)

    • The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.


    The DOM is not part of the JavaScript language, but is instead a Web API used to build websites.

    The DOM API is used directly in JavaScript from within what is called a script, a program run by a browser.

    The DOM is not a programming language, but without it, the JavaScript language wouldn't have any model or notion of web pages, HTML documents, SVG documents, and their component parts


    The document as a whole, the head, tables within the document, table headers, text within the table cells,paragraphs,headings and all other elements in a document are parts of the document object model for that document. They can all be accessed and manipulated using the DOM and a scripting language like JavaScript.

    For example, the following function creates a new <p> element, adds text to that element, and then adds it to the tree for the document:

    <!DOCTYPE HTML>
  <html>
  <head>
    <script>
       window.onload = function() {
  
           // create   <p> element in an otherwise empty HTML page
           const paragraph = document.createElement("p");
    
           //adds content to the created empty <p> element
           const paragraph_text = document.createTextNode("Welcome to web page manipulation using DOM and javascript!");
    
           //attaches the content to the <p> element 
           paragraph.appendChild(paragraph_text);
    
           //attaches the <p> element  to the body of the  HTML document
           document.body.appendChild(paragraph);
        }
    </script>
  </head>
  <body>
  </body>
  </html>
  • The Browser Object Model (BOM)

    • The Browser Object Model (BOM) in JavaScript includes the properties and methods for JavaScript to access and manipulate the browser window. BOM provides you with:

    • The window object to show the size of the window
    • The window.screen object to show the size of the screen
    • The window.location object can be used to get the current page address (URL) and to redirect the browser to a new page..
    • Cookies let you store user information in web pages.
    • Popup boxes: Alert box, Confirm box, and Prompt box.
    <!DOCTYPE HTML>
  <html>
  <head>
    <script>
        document.write("Screen Width: " + screen.width);
        console.log("window.innerHeight");
        document.alert("window.location.href");
           document.write("Screen height: " + screen.height);
           window.confirm("Are you a pop box?");
    </script>
  </head>
  <body>
  </body>
  </html>

    There are varying levels of support for the three parts of JavaScript across the five major web browsers (Internet Explorer, Firefox, Chrome, Safari, and Opera). Support for ECMAScript 3 is generally good across all browsers, and support for ECMAScript 5 is growing, whereas support for the DOM varies widely. The BOM, recently codified in HTML5, can vary from browser to browser

    What Javascript can do
    • Javascript plays an important role in validating user inputs before submitting it to the server.
    • Add new HTML to the page, change the existing content, modify styles.
    • Get and set cookies, ask questions to the visitor, show messages.

    client-side and server-side scripting Language

    JavaScript was initially created as a browser-only language,but today,it can execute on the server,actually on any device that has a special program called the JavaScript engine.

  • Client-side version

    • On the client side,JavaScript is run by v8 engine (Google chrome).Client Side JavaScript enables the enhancement and manipulation of web pages and client browsers. Scripts are embedded in or included from HTML documents and interact with the DOM.

  • Server-side version

    • Server Side JavaScript is an extended version of JavaScript that enables back-end access to databases, file systems, and servers.The v8 engine(Google chrome with some modifications to provide the server functionality) is also used in the servers to run javascript codes.

    Javascript Syntax

    These are set of rules that define a correctly structured JavaScript program

  • JavaScript Identifiers / Names

    • An identifier is the name of a variable, function or property. Identifiers may be one or more characters in the following format:

    • The first character must be a letter, an underscore (_), or a dollar sign ($).
    • All other characters may be letters, underscores, dollar signs, or numbers.

    By convention, ECMAScript identifiers use camel case, meaning that the first letter is lowercase and each additional word is offset by a capital letter, like this:
    firstSecond
    sayHelloWorld

  • Names are case sensitive(y and Y are different Identifiers).
  • Keywords, reserved words, true, false, and null cannot be used as identifiers.

    • NOTE −Javascript identifiers are case-sensitive(Y and y are different)

  • JavaScript Variables

    • Variables are used to store data values. JavaScript uses the keywords var, let and const to declare variables. An equal sign(=),is used to assign values to variables.
    Example:

    let message;

    Now, we can put some data or value into it by using the assignment operator =

    let message;
            message = 'Welcome';

    Multiple variable can also be declared in a single line,separated by a comma: 

    var a = 'welcome', b = 10, c = 23.6;

    Variables can be declared either inside a function or outside a function


    Variables that can be declared inside a function and also accessed inside a function are called Local variable

    Variables that can be declared outside a function and accessed anywhere in the javascript code, even inside a function are called Global Variables.

    Example: 
    
                var globalVariable = "javascript"; //This is a global  variable
                function myFunction(){
                  var  localVariable = "This is a local variable"; //Local variable
                  alert(localVariable + globalVariable) //can be accessed inside the function.
                }
                myFunction();
                alert(globalVariable); //can access the global variable outside the function;
                alert(localVariable); // cannot access the local variable outside the function;
    Learn Global and scope in Javascript

    The string is now saved in the variable. We can access it using the variable name: 

    alert(message);

  • JavaScript is case sensitive.

    • It is common to start the name of a constructor with a capitalised letter, and the name of a function or variable with a lower-case letter.
    So the identifiers Fruit and FRUIT will convey different meanings in JavaScript.

  • Semicolons ;

    • Semicolon is added at the end of each executable statement:

    Examples

    let a, b, c;      // Declares 3 variables
          a = 10;         // Assign the value 10 to a
          b = 20;         //Assign the value 20 to b
          c = a + b;       // Assign the sum of a and b to c

    When separated by semicolons, multiple statements can be written on one line:

    
      a = 5; b = 6; c = a + b;
  • JavaScript Comments

    • JavaScript comments can be used to explain JavaScript code, and to make it more readable.

    A single-line comment begins with two forward-slash characters //

    This example uses a single line comment at the end of each line to explain the code:

    let a = 12;        // Declare a, gives it the value of 12

    A Multi-line comment begins with a forward slash and asterisk (/*) and ends with the opposite (*/).
    This example uses a multi-line comment (a comment block) to explain the code


    /* The code below will add the text in quotes to the elements with id="heading" and id="paragraph"in your HTML document: */

    
  document.getElementById("heading").innerHTML = "Javascript comments";
  document.getElementById("paragraph").innerHTML = "This is javascript multi-line comments";

    NOTE −Any text between // and /* */ will not executed by JavaScript.

To enable strict mode for an entire script, include the following at the beginning of your javascript file: “use strict”; this tells supporting JavaScript engines to change into strict mode.     Javascript Statement

    JavaScript statements are the commands that tell the browser what actions to perform. Actions like

    • Creating a variable or a function
    • Looping through an array of elements
    • Evalauting code based on a specific condition etc.

    Statments in javascript can be grouped into the following:


    • Declaration Statemnts

      • These are statements that create variables and functions by using the var and function statements

    • Conditional Statements

      • They are used to perform different actions based on different conditions

      • The if Statement: specifies a block of code to be executed if a condition is true
        • 
                if(20 >10){
                  console.log("correct")
                }
                // correct shows

        (20 >10) is the condition to test,which in this case is true and because is true, "correct" gets shown

      • Else Statement shows when the if condition is not true
        • 
                if(20 < 10){
                console.log("correct")
                }else{
                console.log("wrong")
                }
                              // wrong shows

        (20 < 10)the condition is not true,so "wrong" gets shown

      • Else if Statement specifies a new condition for testing when the first if condition is "false"
        • 
                      if(20 < 10){
                       console.log("apple")
                        }else if(20 > 10)
                        {
                        console.log("tomato")
                        }
                          // tomato shows since the first condition was wrong

        NOTE −if is in lowercase letters.Uppercase letters (If or IF) will generate a javascript error

      • Switch Statement is a collection of case statements that are compared to the switch condition and evaluated when the condition and case are true.A break is used betweenn the cases to prevent additional execution.A default gets evaluated when none of the cases are true
        • Example: 
        
                  let color ="green";
                  switch(color){
                    case:"orange":
                    console.log('A mix of red and yellow');
                    break;
                    case:"green":
                    console.log('A mix of blue and yellow');
                    break;
                    default:
                    console.log("Not sure about this one");
                    break;
                  }
                  // A mix of blue and yellow shows
      • For loop: used to run code a set number of times
        • 
                for(let i =0; i < 5; i++){
                  console.log(i);
                }
                //1 2 3 4 5 show
                i is set to  0 ;
         i< 5 is checked each time and if the condition is true, i is increased until i = 5,then the code stops running.
      • While statement executes a specified statement as long as the condition evaluates to true
        • 
                let n =0;
                while (n < 3){
                  n++;
                } console.log(n);
                // 3 shows
